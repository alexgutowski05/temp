How To Record and Submit Your Recordings

Windows (preferred) macOS chromeOS (Not Recommended)

Windows



  1. Click Sheet Music on the top. You will be redireted to Sound Slice
  2. Choose which music you would like to record. Your character requires the following songs:
    • This is admin account. No music is required
  3. Use your web browser to go to www.obsproject.com, and download the Windows installer. Follow the setup instructions like you would with a normal program.
  4. Once OBS Studio is installed, launch the program.
  5. In the upper left corner, click File > Settings > Output
  6. On the top, click the dropdown next to "Output Mode" and select "Advanced"
  7. Below the dropdown, select "Recording"
  8. Set "Recording Format" to "mkv", select Audio Tracks 1 and 2, set "Encoder" to x264, set "Rate Control" to "CRF", enter "23" as the "CRF" value, and set "CPU Usage Preset" to "veryfast"
  9. Press "Apply" then "OK"
  10. On the bottom, the second panel from the left is named "Sources". Press the "+" at the bottom of the panel, and select "Video Capture Device"
  11. Name this device "Webcam" and press "OK"
  12. Under "Device" select something similar to "HD Webcam" or "Webcam" then press "OK". The device name is different on every computer.
  13. Next to "Sources", there is the "Audio Mixer". Click the gear next to Mic/Aux and select "Advanced Audio Properties". (If Mic/Aux is not an option, click here
  14. Next to "Desktop Audio", deselect every "Track" except for #1. Do the same with "Mic/Aux" except leave #2 checked instead of #1.
  15. Plug headphones into your computer
  16. Press "Start Recording" on OBS, and press play on the sheet music
  17. Sing your song
  18. When you are finished, press "Stop Recording"
  19. Download MKVToolNix from fosshub and install it.
  20. Launch MKVToolNix
  21. Right click in the "Source Files" box and say "Add Files"
  22. Navigate to your "Videos" folder
  23. Select the file you just recorded. Please only include the recording you want me to use. You may have to use a tool such as 123APPS's Online Video Cutter.
  24. Select "Open"
  25. Under "Tracks, Chapters, and Tags", Deselect everything except for AAC2. If you hover over it, it should say "Track2"
  26. Scroll down and press "Start Multiplexing"
  27. Find the file it created in your "Videos" folder. It will have a .mka file extention.
  28. Visit ConvertFiles, and convert your .mka file to a .mp3 file
  29. Head back to SoundSlice and on the sheet music you just recorded, select "+ Add Performance" and upload the .mp3 file you just downloaded
  30. Log back into the Cast Portal, and on the top bar, select "DEH gitrepo" or click here.
  31. Select "Fork" in the upper right hand corner
  32. Login or create a GitHub account
  33. Wait for the "forking" process to complete
  34. Find where it says "Branch: master", click on it, and choose "portal". It should be on the left, above the files.
  35. Navigate to "music" and push "Upload files". Upload the .mkv file, not the mp3 file or the mka file
  36. Press "Commit Changes" to save.
  37. Next to "Branch: master", which now says "Branch: portal", click "New Pull Request". Select the file upload you just made and select "Send Pull Request"
  38. Sorry. I know that was a lot. Thanks for hanging in.
